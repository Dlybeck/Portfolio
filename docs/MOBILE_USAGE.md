# Mobile Keyboard Usage

The terminal interface includes a special mobile keyboard toolbar that appears automatically on devices smaller than 768px wide.

## Features

- **Esc**: Sends Escape key (vim, canceling commands)
- **Tab**: Sends Tab key (autocompletion)
- **Ctrl**: Toggles Control modifier (turns purple when active)
- **Alt**: Toggles Alt modifier (turns purple when active)
- **Arrows**: Up/Down/Left/Right navigation

## How to Use

1. **Tap Ctrl** to enable it (it stays active).
2. **Tap C** on your virtual keyboard to send `Ctrl+C` (interrupt).
3. **Tap Ctrl** again to disable it.

## Troubleshooting

If the keyboard doesn't appear:
1. Ensure your browser window is narrow enough (< 768px).
2. Refresh the page.
